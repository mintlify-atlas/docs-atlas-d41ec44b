  <h2 className="text-2xl sm:text-3xl font-semibold text-gray-900 dark:text-white mb-4">
    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with torch_brain in minutes
  </p>

  <Steps>
    <Step title="Install the package">
      Install torch_brain using pip in your Python environment (Python ≥ 3.10 required):

      ```bash
      pip install torch_brain
      ```

      For development, install with optional dependencies:

      ```bash
      pip install -e ".[dev]"
      ```
    </Step>

    <Step title="Prepare your neural data">
      torch_brain works with neural recordings from multiple sessions. Use the companion [brainsets](https://github.com/neuro-galaxy/brainsets) library to access curated datasets:

      ```bash
      brainsets prepare perich_miller_population_2018
      ```

      This downloads and prepares a multi-session motor cortex dataset ready for training.
    </Step>

    <Step title="Train a model">
      Train a POYO model for neural population decoding:

      ```python
      import torch
      from torch_brain.models import POYO
      from torch_brain.data import Dataset

      # Load your neural data
      dataset = Dataset(
          data_path="path/to/neural/recordings",
          modalities=["spikes", "behavior"]
      )

      # Initialize POYO model
      model = POYO(
          sequence_length=2.0,
          dim=256,
          num_latents=128,
          num_self_attn_layers=6
      )

      # Train with PyTorch Lightning or your training loop
      # See the guides for complete training examples
      ```

      <Accordion title="Expected output">
        The model will process spike trains from multiple neurons and sessions, learning a unified representation for downstream decoding tasks.
      </Accordion>
    </Step>
  </Steps>
