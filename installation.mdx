torch_brain is available for Python >= 3.10 and can be installed via pip.

## Prerequisites

<Note>
  torch_brain requires Python 3.10 or higher. We recommend using Python 3.10 for best compatibility.
</Note>

Before installing torch_brain, ensure you have:

- Python 3.10 or higher
- pip (Python package installer)
- A virtual environment manager (recommended)

## Basic installation

### Step 1: Create a virtual environment

We recommend using a virtual environment to manage dependencies:

<CodeGroup>

```bash venv
python3 -m venv venv
source venv/bin/activate  # On Windows, use: venv\Scripts\activate
```

```bash conda
conda create -n torch_brain python=3.10
conda activate torch_brain
```

</CodeGroup>

### Step 2: Install torch_brain

Once your virtual environment is activated, install torch_brain:

```bash
pip install torch_brain
```

This will install torch_brain and its core dependencies:

- `temporaldata>=0.1.3` - For handling temporal neuroscience data
- `torch~=2.0` - PyTorch deep learning framework
- `einops~=0.6.0` - Flexible tensor operations
- `hydra-core~=1.3.2` - Configuration management
- `torchtyping~=0.1` - Type annotations for tensors
- `torchmetrics>=1.6.0` - Metrics for model evaluation
- `pydantic~=2.0` - Data validation
- `rich` - Beautiful terminal output

### Step 3: Verify installation

Verify that torch_brain is installed correctly:

```python
import torch_brain
from torch_brain.models.poyo import POYO
print("torch_brain installed successfully!")
```

## Development installation

If you plan to contribute to torch_brain or modify the source code, install it in development mode:

<Steps>

<Step title="Clone the repository">

```bash
git clone https://github.com/neuro-galaxy/torch_brain.git
cd torch_brain
```

</Step>

<Step title="Install in editable mode with dev dependencies">

```bash
pip install -e ".[dev]"
```

This installs additional development tools:
- `pytest<9` - Testing framework
- `black==24.2.0` - Code formatting
- `pre-commit>=3.5.0` - Git hooks for code quality
- `flake8` - Linting
- `pytest-cov` - Code coverage
- `brainsets` - Access to neuroscience datasets

</Step>

<Step title="Install pre-commit hooks">

```bash
pre-commit install
```

This ensures code quality checks run automatically before each commit.

</Step>

<Step title="Run tests">

Verify everything is working by running the test suite:

```bash
pytest
```

To test individual files:

```bash
pytest test/test_binning.py
```

</Step>

</Steps>

## Installing datasets

To train POYO models, you need to install the `brainsets` package for dataset access:

```bash
pip install brainsets
```

Then prepare datasets for training:

<CodeGroup>

```bash Single dataset
brainsets prepare perich_miller_population_2018
```

```bash Multiple datasets
brainsets prepare perich_miller_population_2018
brainsets prepare churchland_shenoy_neural_2012
brainsets prepare flint_slutzky_accurate_2012
brainsets prepare odoherty_sabes_nonhuman_2017
```

</CodeGroup>

See the [brainsets documentation](https://brainsets.readthedocs.io/) for more information on available datasets.

## GPU support

torch_brain automatically detects and uses GPUs when available. For GPU acceleration:

1. Install CUDA-compatible PyTorch:

```bash
pip install torch --index-url https://download.pytorch.org/whl/cu118
```

2. Verify GPU is detected:

```python
import torch
print(f"CUDA available: {torch.cuda.is_available()}")
print(f"GPU count: {torch.cuda.device_count()}")
```

## Additional dependencies

For training with logging and experiment tracking:

```bash
pip install lightning wandb
```

- `lightning` - PyTorch Lightning for training
- `wandb` - Weights & Biases for experiment tracking

## Troubleshooting

<AccordionGroup>

<Accordion title="ImportError: No module named 'torch_brain'">
  Ensure torch_brain is installed in your current Python environment:
  ```bash
  pip list | grep torch_brain
  ```
  If not listed, reinstall with `pip install torch_brain`.
</Accordion>

<Accordion title="Version conflicts with PyTorch">
  torch_brain requires PyTorch 2.0 or higher. Update PyTorch:
  ```bash
  pip install --upgrade torch
  ```
</Accordion>

<Accordion title="CUDA/GPU issues">
  Verify your CUDA installation matches your PyTorch version. See the [PyTorch installation guide](https://pytorch.org/get-started/locally/) for CUDA-specific installation instructions.
</Accordion>

</AccordionGroup>
