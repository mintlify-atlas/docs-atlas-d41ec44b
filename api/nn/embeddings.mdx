## Embedding

A simple extension of `torch.nn.Embedding` with better control over weight initialization.

### Class Signature

```python
class Embedding(nn.Embedding):
    def __init__(
        self,
        num_embeddings: int,
        embedding_dim: int,
        init_scale: float = 0.02,
        **kwargs,
    )
```

### Parameters

<ParamField path="num_embeddings" type="int" required>
  Size of the dictionary of embeddings
</ParamField>

<ParamField path="embedding_dim" type="int" required>
  The size of each embedding vector
</ParamField>

<ParamField path="init_scale" type="float" default="0.02">
  Standard deviation of the normal distribution used for initialization. Defaults to 0.02, which is the standard value used in most transformer models.
</ParamField>

<ParamField path="**kwargs" type="dict">
  Additional arguments passed to `torch.nn.Embedding`. See PyTorch documentation for details.
</ParamField>

### Description

The learnable weights of shape `(num_embeddings, embedding_dim)` are initialized from a normal distribution $\mathcal{N}(0, \text{init_scale})$.

### Shape

- Input: `(*)` - LongTensor of arbitrary shape containing indices
- Output: `(*, embedding_dim)` - Embedding vectors

### Usage Example

```python
from torch_brain.nn import Embedding
import torch

# Create embedding layer for vocabulary of 1000 words
embedding = Embedding(
    num_embeddings=1000,
    embedding_dim=128,
    init_scale=0.02
)

# Embed some token indices
input_ids = torch.LongTensor([[1, 2, 3], [4, 5, 6]])
embedded = embedding(input_ids)
print(embedded.shape)  # torch.Size([2, 3, 128])
```

## InfiniteVocabEmbedding

An embedding layer with a dynamically extendable vocabulary. This module acts as both an embedding layer and a tokenizer, making it ideal for datasets with dynamically generated vocabularies.

### Class Signature

```python
class InfiniteVocabEmbedding(nn.Module):
    def __init__(
        self,
        embedding_dim: int,
        init_scale: float = 0.02
    )
```

### Parameters

<ParamField path="embedding_dim" type="int" required>
  Embedding dimension
</ParamField>

<ParamField path="init_scale" type="float" default="0.02">
  Standard deviation of the normal distribution used to initialize the embedding matrix
</ParamField>

### Key Features

- **Lazy initialization**: Weight matrix is created when vocabulary is initialized or loaded from checkpoint
- **Dynamic vocabulary**: Can extend vocabulary with new words or subset to selected words
- **Built-in tokenizer**: Provides `tokenizer()` and `detokenizer()` methods
- **Checkpoint compatibility**: Vocabulary is saved/loaded with model state

### Shape

- Input: `(*)` - LongTensor of token indices
- Output: `(*, embedding_dim)` - Embedding vectors

### Methods

#### initialize_vocab()

```python
def initialize_vocab(self, vocab: List[str])
```

Initialize the vocabulary with a list of words. Should be called only once before training.

<ParamField path="vocab" type="List[str]" required>
  List of words to initialize the vocabulary. A special word "NA" is automatically added at index 0 for padding.
</ParamField>

<Note>
  This method should be called only once. To add new words after initialization, use `extend_vocab()` instead.
</Note>

#### extend_vocab()

```python
def extend_vocab(self, vocab: List[str], exist_ok: bool = False)
```

Extend the vocabulary with new words. Embeddings for new words are initialized randomly.

<ParamField path="vocab" type="List[str]" required>
  List of words to add to the vocabulary
</ParamField>

<ParamField path="exist_ok" type="bool" default="False">
  If True, skip words that already exist without raising an error
</ParamField>

#### subset_vocab()

```python
def subset_vocab(self, vocab: List[str], inplace: bool = True)
```

Select a subset of the vocabulary. Embeddings for selected words are copied from the original.

<ParamField path="vocab" type="List[str]" required>
  List of words to select from the vocabulary
</ParamField>

<ParamField path="inplace" type="bool" default="True">
  If True, modify the vocabulary in place. If False, return a new InfiniteVocabEmbedding.
</ParamField>

#### tokenizer()

```python
def tokenizer(self, words: Union[str, List[str]]) -> Union[int, List[int]]
```

Convert words to token indices.

<ParamField path="words" type="Union[str, List[str]]" required>
  A word or list of words to tokenize
</ParamField>

Returns the token index (int) or list of token indices (List[int]).

#### detokenizer()

```python
def detokenizer(self, index: int) -> str
```

Convert a token index to a word.

<ParamField path="index" type="int" required>
  Token index to convert
</ParamField>

#### is_lazy()

```python
def is_lazy(self) -> bool
```

Returns True if the module has not been initialized yet.

### Usage Example

```python
from torch_brain.nn import InfiniteVocabEmbedding
import torch

# Create embedding layer (initially lazy)
embedding = InfiniteVocabEmbedding(embedding_dim=64)

# Initialize vocabulary
vocab = ["apple", "banana", "cherry"]
embedding.initialize_vocab(vocab)

print(embedding.vocab)
# OrderedDict([('NA', 0), ('apple', 1), ('banana', 2), ('cherry', 3)])

print(embedding.weight.shape)
# torch.Size([4, 64])

# Tokenize words
token_ids = embedding.tokenizer(["apple", "cherry", "banana"])
print(token_ids)  # [1, 3, 2]

# Get embeddings
token_tensor = torch.tensor(token_ids)
embedded = embedding(token_tensor)
print(embedded.shape)  # torch.Size([3, 64])

# Extend vocabulary
embedding.extend_vocab(["date", "elderberry"])
print(embedding.weight.shape)  # torch.Size([6, 64])

# Subset vocabulary
embedding.subset_vocab(["apple", "banana"])
print(embedding.vocab)
# OrderedDict([('NA', 0), ('apple', 1), ('banana', 2)])
```

### Loading from Checkpoint

<Warning>
  If you want to load only a subset of words from a checkpoint, do NOT call `initialize_vocab()` before loading. Instead, load the checkpoint first, then use `subset_vocab()`.
</Warning>

```python
# Load checkpoint first, then subset
embedding = InfiniteVocabEmbedding(embedding_dim=64)
embedding.load_state_dict(checkpoint['embedding'])
embedding.subset_vocab(["word1", "word2"])  # Only keep these words
```
